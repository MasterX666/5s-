修改题库.ini文件，修改请注意格式，避免不必要的麻烦。  
格式参考（带“//“表示必须含有）：  
[用“不行”造一句 并 包含“不要”且少于 8 个字的句子！]//  
long=3  
size=8  
类型=4//  
题目=用“不行”造一句 并 包含“不要”且少于 8 个字的句子！//  
type1=1  
text1=不行  
type2=1  
text2=不要  
  
类型说明：  
1：任意回复  
2：任意qq经典表情  
3：任意qq大表情（包括图片）  
4：文字相关  
     long：表示文字相关类型  
               long=1：大于几个字符  
               long=2：等于几个字符  
               long=3：小于几个字符  
               long=4：不等于几个字符  
     size:表示文字长度  
     text1、text2为关键词  
     type1、type2=1：表示必须包含 为0则表示不能包含  
6：at任意人  
7：回复指定内容  
      答案：该指定内容
